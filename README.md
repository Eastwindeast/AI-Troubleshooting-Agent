# AI-Driven Autonomous Troubleshooting Agent

这是一个基于大模型（LLM）和长链推理（Reasoning Loop）构建的**智能排障 Agent**。它模拟了 SRE 工程师在面对线上故障时的标准排查路径：**接收告警 -> 检索日志 -> 关联代码变更 -> 深度根因分析**。

# 核心
- **自主决策**：根据告警上下文自动判断是否需要查看日志或代码，而非死板的脚本。
- **长链推理**：跨越日志系统与 Git 仓库的数据孤岛，进行多维度的关联分析。
- **闭环反馈**：不仅仅输出错误原因，更直接给出可执行的修复方案（如回滚或代码补丁）。
